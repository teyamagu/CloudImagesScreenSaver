# Security

## Credentials

This screen saver stores your Dropbox **access token** in macOS `ScreenSaverDefaults` for the module. Treat it like a password.

- **Do not** paste tokens, plist dumps, or screenshots containing tokens into GitHub issues or pull requests.

## Reporting a security issue

If you find a vulnerability in this project’s code (for example, unsafe handling of tokens or network data), please report it **privately** to the repository maintainers (e.g. GitHub Security Advisories / private disclosure, if enabled on the repo) rather than opening a public issue first.

This project is **not** affiliated with Dropbox, Inc. For issues with Dropbox’s API or your account, use [Dropbox support](https://help.dropbox.com/).
